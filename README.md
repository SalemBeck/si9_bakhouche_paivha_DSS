 Application Trains

 Comment démarrer

 1. Installer Flask
```
pip install flask
```

 2. Lancer l'application
```
python app.py
```

 3. Ouvrir dans le navigateur
Va sur `http://localhost:5000`


- Tous les trajets : Affiche tous les trajets disponibles
- Rechercher : Cherche un trajet par son code
- Filtrer : Filtre par départ, arrivée, type ou prix
- Statistiques : Voir les prix min/max par ligne
